# Healthcare ETL Pipeline

## Overview
Extract healthcare data from public API, transform with Python/pandas, and load to PostgreSQL database on AWS RDS.

## Status
🚧 In Development

## Tech Stack
- Python 3.x
- pandas
- requests
- psycopg2
- PostgreSQL (AWS RDS)

## Structure
- `extract.py` - API data extraction
- `transform.py` - Data cleaning and transformation  
- `load.py` - Database loading
